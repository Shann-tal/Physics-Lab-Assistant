Physics Lab Assistant - 大物实验小帮手

Physics Lab Assistant 是一个专为大学物理实验课程设计的开源桌面应用。它旨在成为学生的智能实验伙伴，通过简化数据处理、自动计算和可视化，帮助学生专注于理解实验原理与现象，而非繁琐的计算过程。

项目愿景

我们认识到，学生在物理实验中常常耗费大量时间在重复性的数据计算和图表绘制上，这反而削弱了对实验核心物理思想的探索。本项目致力于解决这一痛点，打造一个轻量、智能、完全开放的实验工具，让技术赋能教育，提升学习效率与深度。

核心功能

(1)实验预习向导：提供清晰的实验目的、原理和步骤指南，帮助学生提前构建知识框架，带着问题进入实验室。
(2)智能数据计算：实验结束后，用户只需输入测量的原始数据，程序将自动调用预置或自定义的算法，完成全部中间结果与最终结果的计算，并给出完整计算过程。
(3)动态图表生成：支持散点图、线性拟合、曲线图等多种可视化。如果实验涉及变量关系，系统能一键生成并美化图表，使物理规律一目了然。
(4)透明计算过程：不仅给出答案，更展示每一步的计算公式与逻辑。这确保了"黑箱"疑虑的消除，并使其成为一个绝佳的学习参考工具。
(5)开源与可扩展：作为开源软件，鼓励教师和学生根据本校的具体实验要求，修改、添加新的实验模块与算法，共同构建一个更全面的实验库。

技术栈

项目采用简单易学的C++技术栈，非常适合初学者入门：

核心库

· Qt Framework - 图形界面开发，文档丰富、学习资源多
· 标准C++库 - 使用STL完成基础数据结构和算法

数据处理

· 自己实现计算函数 - 亲手编写物理公式计算逻辑
· CSV文件读写 - 使用标准文件操作处理数据输入输出

图表绘制

· Qt Charts - Qt自带的图表模块，简单易用
· 控制台输出图表 - 使用字符方式简单展示数据趋势

开发工具

· GCC/G++编译器 - 开源免费的编译工具
· VSCode - 轻量级开发环境


开源协议

本项目采用MIT开源协议，鼓励学术使用和二次开发。
